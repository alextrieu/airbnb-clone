## Introduction
This project allowed me to work on several important aspects of modern web development, including:

Creating reusable and modular components: To create the different parts of the Airbnb website, I broke the UI down into individual components that could be reused throughout the application. This helped to keep the code organized and made it easier to modify specific parts of the website.

Generating dynamic content with array mapping: To display a list of properties on the website, I used the Array.map() function to generate the necessary HTML elements dynamically. This allowed me to add new properties to the site without having to hard-code each one individually.

Navigating between pages: To create a seamless user experience, I used React Router to handle navigation between different parts of the site. This allowed users to move from page to page without having to reload the entire site.

Managing environment variables and APIs: To keep the site secure and efficient, I used environment variables to store sensitive data and accessed APIs to retrieve dynamic content. This included rendering a Google Map component using an NPM package and using environment variables to securely store the Google Maps API key. This approach helped to ensure that the site remained responsive, user data was protected, and the Google Maps API was accessed securely.

Implementing modals: To create a more interactive user experience, I used modals to display additional information about properties and other features on the site. This made the site more engaging and helped users to find the information they needed quickly.

Creating a mobile responsive website: To ensure that the site was accessible and usable on a range of devices, I made the site responsive to different screen sizes. This helped to ensure that users could access the site from their desktops, tablets, and smartphones.

Overall, building the Airbnb clone was an excellent exercise in modern web development and helped me to develop my React skills in a number of important areas. The project challenged me to think critically about the user experience and taught me how to build modular, responsive, and engaging web applications.

## Installation & Usage 

Ensure [Node.js](https://nodejs.org/en/) and [NPM](https://www.npmjs.com/) are installed

1. Clone or Download the repository (Depending on whether you are using SSH or HTTPS)
```
git clone git@github.com:alextrieu/airbnb-clone.git
cd airbnb-clone
```
2. Install dependencies
```
npm i
```
3. Start the application
```
npm start
```

After the application starts visit http://localhost:3000 to view it in the browser.
